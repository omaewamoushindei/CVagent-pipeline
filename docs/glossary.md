# Glossary

## Raw Notes

Unstructured source material such as career notes, project descriptions, prior CV content, or achievement drafts.

## Career Vault

The structured source of truth for reusable career information. It organizes experience, education, projects, skills, metrics, and supporting evidence.

## Master CV

A reusable base CV created from the Career Vault. It is not tied to one job description.

## Target Job Description

The role-specific input used to identify keywords, responsibilities, requirements, and positioning signals.

## Tailored Resume

A job-specific resume variant generated from the Master CV and selected Career Vault evidence.

## Prompt Step

A documented instruction file that guides one part of the workflow. Prompt files are version-controlled so changes can be reviewed and improved.

## Evaluation Rubric

A documented quality framework used to review outputs for relevance, evidence consistency, metric density, formatting, and hallucination risk.

## Benchmark Dataset

Synthetic test data used to evaluate prompt and workflow changes without exposing private information.

## Generated Output

A document created by the workflow, such as a tailored resume, cover letter, keyword analysis, or rendered PDF.

## Source of Truth

The primary location where reliable information is maintained. In this project, the Career Vault and PLAYBOOK serve as the main sources of truth.
