# Examples

This folder contains synthetic demo material for reviewing the workflow without using private career data.

## Contents

| Path | Purpose |
|---|---|
| `synthetic-raw/sample.md` | Example raw notes input (synthetic) |
| `synthetic-job-description.txt` | Example target job description |
| `synthetic-career-vault/` | Example structured source content |
| `sample-output/` | Example generated artifacts |

## Usage

Use this folder to understand what the workflow can produce before running it with your own data.

For a new personal run, use the local workspace folders:

- `raw/`
- `career_vault/`
- `applications/`

Do not add private personal data to this folder.
